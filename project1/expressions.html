Expressions

Expression vs Equations

An expression and equation look the same at first glance. They both contain numbers, operations and/or variables. The best way to differentiate if it is an expression or an equation is if it contains an equal sign (=). If it has an equal sign, it is an equation, if it does not it is just an expression.

Expression
– A phrase that contains numbers, letters with operators to represent a value.

Equation
– A statement of equality. It contains 2 expressions connected by an equal sign.

2 + 3; is an expression
2 + 3 = 9 – 4; is an equation

Variable
– A letter that represents a number wherein it’s value may change.

Constant
– A number where it’s value never changes and always stays the same.

Inequality
– An expression comparing two values with the usage of various inequality symbols (≠, <, ≤, >, ≥).

Grouping Symbols
– Used to determine which expressions are to be kept together. More will be discussed during the lesson on PEMDAS and Order of Operations.

Numerical Expressions
– Expressions that contain numbers and operators without any unknown variables or inequality symbols.

Algebraic Expressions
– Expressions that contain numbers and operators as well as variables without inequality symbols.

Examples

  • 40 + 80 is a numerical expression
  • 5x + 9 is an algebraic expression
  • 30 + 5 = 45 – 10 is an equation
